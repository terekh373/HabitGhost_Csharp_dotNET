# HabitGhost — Менеджер звичок та продуктивності

> Десктопний Windows-застосунок для відстеження звичок, цілей та аналітики продуктивності.  
> Розроблений на C# / WPF / .NET як підсумкова курсова робота — **1-е місце на потоці академії**.

---

## Про проєкт

HabitGhost допомагає користувачам формувати корисні звички, ставити цілі та відстежувати прогрес. Застосунок отримує та передає дані через TCP-з'єднання на SQL Server, працює у фоновому режимі через трей-іконку та підтримує автозапуск при старті Windows.

---

## Функціонал

- **Вкладка Звички** — створення, редагування, відстеження щоденних звичок
- **Вкладка Цілі** — постановка цілей з дедлайнами та прогрес-барами
- **Вкладка Аналітика** — графіки виконання звичок (щоденно / щотижнево), відсоток виконаних цілей
- **Вкладка Журнал** — історія активності
- **Вкладка Повідомлення** — непрочитані сповіщення
- **Трей-іконка** — контекстне меню, прихована робота у фоні
- **Налаштування** — темна/світла тема, автозапуск, керування сповіщеннями
- **Авторизація** — реєстрація та вхід користувача

## Запуск проєкту

### Вимоги
- Windows 10/11
- .NET 6.0 або вище
- SQL Server (локальний або мережевий)
- Visual Studio 2022

### Кроки
```bash
# 1. Клонувати репозиторій
git clone https://github.com/terekh373/HabitGhost_Csharp_dotNET.git

# 2. Відкрити у Visual Studio
# Відкрий файл HabitGhost_Repo/HabitGhost.sln

# 3. Налаштувати рядок підключення до БД
# Знайди appsettings або конфіг-файл і вкажи свій SQL Server

# 4. Запустити проєкт
# Натисни F5 або кнопку Run у Visual Studio
```
