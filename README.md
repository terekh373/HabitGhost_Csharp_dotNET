## 📖 About

HabitGhost helps users build healthy habits, set goals, and track progress over time. The app communicates with a SQL Server database via TCP, runs silently in the background through a system tray icon, and supports auto-launch on Windows startup.

## ✨ Features

### 📋 Core Tabs
- **Habits** — create, edit, and track daily habits with streaks
- **Goals** — set goals with deadlines and progress bars
- **Analytics** — habit completion charts (daily / weekly), goal completion percentage
- **Journal** — full activity history log
- **Notifications** — unread alerts and reminders

### ⚙️ System Features
- 🌙 **Dark / Light theme** — toggle in settings
- 🔄 **Auto-launch** — starts with Windows automatically
- 🖥️ **System tray icon** — runs hidden in background with context menu
- 🔔 **Smart notifications** — reminders for habits and goals
- 🔐 **User authentication** — registration and login system

## 🚀 Getting Started

### Prerequisites
- Windows 10 / 11
- .NET 6.0 or higher
- SQL Server (local or remote)
- Visual Studio 2022

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/terekh373/HabitGhost_Csharp_dotNET.git

# 2. Open in Visual Studio
# Open: HabitGhost_Repo/HabitGhost.sln

# 3. Configure database connection
# Find appsettings or config file and set your SQL Server connection string

# 4. Run the project
# Press F5 or click Run in Visual Studio
```
